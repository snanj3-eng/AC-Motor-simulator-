Induction Motor Explorer (Single HTML File)

This simulator is designed for your main goal: clearly understanding the induction motor principle. Focus on this chain: rotating stator field → slip → induced rotor current → torque.

Key truth: if slip becomes zero, induced rotor EMF becomes zero, so electromagnetic torque disappears.

What changed for realism

  • Rotor speed is no longer hard-locked; it accelerates/decelerates using dω/dt = (Te - Tl - Bω)/J.
  • Electromagnetic torque depends on slip with a simplified induction-motor shape.
  • Pole count affects synchronous speed and the number of air-gap flux lobes in animation.
  • Load slider directly changes mechanical load torque, which changes steady slip in auto mode.

    • How to understand induction motors practically

    1. Start with Preset: Start-up and observe high slip and strong induced bars.
    2. Switch to Preset: Rated; slip drops and operation becomes efficient.
    3. Switch to Preset: Light load; slip approaches zero and torque drops.
    4. Increase poles at fixed frequency and verify Ns = 120f / P decreases.
    5. Increase frequency at fixed poles and observe both Ns and Nr increase.
    • Core equations: Ns = 120f/P, Nr = Ns(1-s), fr = s·f
    • Main principle: torque requires slip.

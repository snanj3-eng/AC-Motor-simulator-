Induction Motor Explorer (Single File)

Improved realism: rotor speed now responds dynamically to electromagnetic torque vs load torque. Animation depends directly on frequency, poles, load, inertia, and slip behavior.

Main principle: no slip -> no induced rotor current -> no electromagnetic torque.

What changed for realism

  • Rotor speed is no longer hard-locked; it accelerates/decelerates using dω/dt = (Te - Tl - Bω)/J.
  • Electromagnetic torque depends on slip with a simplified induction-motor shape.
  • Pole count affects synchronous speed and the number of air-gap flux lobes in animation.
  • Load slider directly changes mechanical load torque, which changes steady slip in auto mode.
